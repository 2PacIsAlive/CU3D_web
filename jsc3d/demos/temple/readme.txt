Download the model of an Ancient Greek Temple. The model and its surrounding environment is in Wavefront Object format (OBJ) and comes complete with textures and materials. The model is part of the rendering demo of the Brilliance framework and was created by Georgios Papaioannou, based on photographic material from the Delphi archaeological site and architectural studies. If you use this mesh for any kind of test or rendering, please acknowledge the creator and provide a link to this page.
